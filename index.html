Calculadora de Partidas Rankeadas

Um projeto de Gabriel Demetrios Lafis

Bem-vindo à Calculadora de Partidas Rankeadas!

Esta ferramenta simples e eficaz foi desenvolvida para ajudar jogadores a calcular seu saldo de vitórias e determinar seu nível (Ferro, Bronze, Prata, Ouro, Diamante, Lendário, Imortal).

Acessar a Calculadora

Funcionalidades Principais:

  • Cálculo de Saldo de Vitórias
  • Determinação de Nível (Rank)
  • Interface Intuitiva
  • Desenvolvido com JavaScript, HTML e CSS
